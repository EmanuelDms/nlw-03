## Como utilizar?
* Clone o repositório;
* Entre na pasta da API e do ambiente desejado:
  * `cd api/` | `cd web/` ou `cd mobile/`
* Instale as dependências em cada uma das duas
  * `yarn install` ou `npm i`
* Use através do comando `yarn start` ou `npm start` !
