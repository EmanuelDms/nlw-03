## Como utilizar?
* Clone o repositório;
* Entre na pasta do ambiente desejado:
  * ` cd web/` ou `cd mobile/`
* Instale as dependências
  * `yarn install` ou `npm i`
* Use através do comando `yarn start` ou `npm start` !
